A responsive, web-based scientific calculator built using **Python 3.10+** and **Streamlit**.

## Features
- **Basic Arithmetic:** Addition, Subtraction, Multiplication, Division, and Powers.
- **Advanced Functions:** Trigonometry, Logarithms, and Square Roots.
- **Responsive UI:** Dynamic layout that works on desktop and mobile.

## How to Run Locally
1. Clone the repo: `git clone https://github.com/umer1556/scientific-calculator.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`
